# Больше / Меньше
Компьютер загадывает случайное число от 0 до 100, пользователь должен вводить свои предположения, а программа отвечать - больше или меньше нужное пользователю число

Компьютер загадал 69

| Ввод от пользователя | Вывод  |
| -------------------- | ------ |
| 50                   | больше |
| 75                   | меньше |
| 69                   | угадал |
Вот функциия для того, чтобы получить случайное число
```js
function randomInt(min, max) {
    return Math.floor(Math.random() * (max - min + 1)) + min;
}
```
